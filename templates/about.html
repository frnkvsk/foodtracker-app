About Food Tracker

Food Tracker is one of my school projects for the Springboard Software Engineering Career Track.

What this app does is allows a user to track their diet intake of nutrients. All the data was obtained exclusively from the USDA FoodData Central API.

Users can create an account and save their menu to use in the future for comparison. Creating an account is not neccessary and a Guest account is set up for anyone to use without signing in.

Getting Started

  1. Enter menu item in grams.
  2. Click Add Ingredient button.
  3. Each menu item can be deleted if needed.

Saving A Menu

  1. Click Save Recipe button.
  2. Add a menu title.
  3. Click Save Recipe button.
  4. Each saved menu can be deleted if needed.
